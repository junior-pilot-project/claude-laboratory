# ⚔️ Reforge

장비를 강화해 **강화합 40**을 달성하는 웹 기반 강화 시뮬레이션 게임.

## 플레이 방법

`index.html`을 브라우저에서 직접 열면 바로 실행된다. 서버·빌드 도구 불필요.

## 게임 흐름

```
골드 획득 → 장비 구매 → 강화 시도 → 레이드 도전 → 골드 획득 → ...
```

**목표**: 칼·방패 강화합의 합이 40이 되면 클리어.

## 주요 기능

### 강화 시스템
- 장비 등급: **하급 / 중급 / 상급** (등급별 기본 스탯·강화 확률 상이)
- Easy Zone (확률 > 30%)과 Hard Zone (확률 ≤ 30%)으로 비용 구조가 달라짐
- 강화 실패 시 장비 파괴

### 랜덤박스
- 다양한 등급의 상자에서 골드 보상 획득
- 개수 지정 연속 오픈 지원

### 레이드
자동 전투로 골드를 대량 수급하는 핵심 콘텐츠.

| 스테이지 | 보스 | 잠금 조건 | 보상 |
|---------|------|---------|------|
| 하급 | 고블린 | 없음 | 30,000G |
| 중급 | 오크 | 강화합 10 | 100,000G |
| 상급 | 데몬 | 강화합 20 | 300,000G |

Canvas로 렌더링된 전투 애니메이션 제공:
- 플레이어 기사: 장착 장비 등급에 따라 외형 변화
- 보스별 고유 공격 모션 (고블린 할퀴기 · 오크 도끼 스윙 · 데몬 화염구)

### 랭킹
클리어 시 플레이 시간이 기록되며 최단 시간 순으로 정렬.

## 기술 스택

- Vanilla JavaScript (ES5 전역 모듈 패턴)
- Canvas 2D API (레이드 전투 시각화)
- Web Audio API (절차적 사운드 생성, 외부 파일 없음)
- localStorage (게임 상태 영속화)
- 외부 라이브러리 없음

## 파일 구조

```
reforge/
├── index.html
├── js/
│   ├── config.js          # 모든 게임 상수 (밸런스 조정은 여기서만)
│   ├── gameState.js       # localStorage 읽기/쓰기
│   ├── itemSystem.js      # 강화 · 장착 · 구매 로직
│   ├── gamblingSystem.js  # 랜덤박스 로직
│   ├── rankingSystem.js   # 랭킹 저장/정렬
│   ├── soundManager.js    # Web Audio API 사운드
│   ├── raidSystem.js      # 전투 시뮬레이션
│   ├── raidCanvas.js      # Canvas 렌더링 · 애니메이션
│   ├── uiManager.js       # DOM 렌더링
│   └── main.js            # 이벤트 핸들러 · 게임 루프
└── style.css
```
