⚡ Race Condition

락 없이 동시 접근 시 2명 이상이 쿠폰을 획득할 수 있음. 심각한 경우 쿠폰 수가 음수가 됨!

🔒 비관적 락

ReentrantLock으로 한 번에 한 스레드만 접근 가능. 안전하지만 대기 시간 발생.

🔄 낙관적 락
