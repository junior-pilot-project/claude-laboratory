# 게임명: 쥐뢰찾기
- 10*10 사이즈로 생성.


# 쥐뢰찾기 규칙
- 10x10 격자에 10개의 지뢰가 랜덤하게 배치됨
- 좌클릭으로 셀을 열기
- 우클릭으로 깃발 표시/제거
- 숫자는 주변 8칸에 있는 지뢰의 개수를 표시
- 지뢰가 없는 빈 칸을 클릭하면 인접한 빈 칸들이 자동으로 열림
- 모든 지뢰가 아닌 칸을 열면 승리
- 지뢰를 클릭하면 게임 오버
- 남은 지뢰 개수 표시
- 게임 시간 측정
- 리셋 버튼으로 새 게임 시작
